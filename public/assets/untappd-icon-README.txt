UNTAPPD ICON PLACEHOLDER
========================

This folder is where the Untappd icon should be placed.

Expected filename:  untappd-icon.png   (preferred)
Alternatives:       untappd-icon.jpg
                    untappd-icon.svg

The server checks for these files in the order listed above
and uses the first one it finds.

HOW TO GET THE ICON
-------------------
1. Download the official Untappd badge/icon from:
      https://untappd.com/media-kit
   or use your own square PNG logo.

2. Recommended specs:
      - Square image (1:1 ratio)
      - At least 200 × 200 px
      - PNG with transparency preferred
      - The icon will be scaled to ~22% of the QR code size

3. Drop the file into this folder as:
      public/assets/untappd-icon.png

4. Restart the server — no code changes needed.

NOTE
----
If this file does NOT exist, the server will still generate QR
codes — they just won't have a centre icon.
